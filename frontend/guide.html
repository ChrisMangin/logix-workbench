Logix Workbench — User Guide

Studio 5000-style editor & comparison tool for Allen-Bradley .L5X files

🚀 Getting Started

  1. Click + Editor in the tab bar to open an editor tab.
  2. Click Open in the toolbar — a native file explorer window opens. Browse to your .L5X file and select it. No upload needed; the server reads directly from disk.
  3. The Controller Organizer sidebar shows your project tree — click any item to view or edit it.
  4. Click + Compare to open a comparison tab. Click the File A and File B slots to browse each file, then click Compare ⇄.
  5. Recent files on the Home screen are clickable — use the filter box to narrow by name or path.
Tip: Multiple + Editor tabs let you work on several files at once. Switch between them with Ctrl+Tab.

🗂 Editor

  • Controller Organizer — Left sidebar. Use - Collapse All / + Expand All buttons at the top to manage the tree. Click any node to open it in the content pane.
  • Tags — View tags by type (Base, Alias, Produced, Consumed). Click ▶ on any expandable tag (UDT, DINT, array) to drill into members or bits. Right-click any row to Copy name or Copy value to clipboard.
  • Ladder Rungs — Visual diagram mode with click-to-edit. Double-click any instruction or tag name to edit inline. Switch to Raw text mode via the mode buttons. Use Ctrl+G to jump directly to a rung number.
  • AOIs — Opens in a tabbed view: Parameters, Local Tags, Logic. Parameters show Default value; logic is fully editable.
  • Data Types — View the full member table. Edit members inline or add new ones.
  • I/O Configuration — View all modules and their connection properties.
  • Tasks & Programs — Browse scheduled programs, their tags, and routines from the task tree.
  • Save — Downloads the modified .L5X back to your computer. Use Close to unload the current file.

⇄ Compare

  • Click + Compare, select File A (baseline) and File B (revised), then click Compare ⇄.
  • The left panel shows a diff tree of every change. Use the filter box to search by name. Click any item to render it side-by-side in File A and File B panes.
  • Find (🔍 icon / toolbar search) in compare mode searches rung text, tag names, routine names, data type names, and AOI names across the entire diff.
  • When a find result is selected, the tool navigates to the correct routine and scrolls to the exact rung in the A or B pane where the match was found.
  • Click 📋 Copy Diff in the Changes header to copy a plain-text summary of all changes to the clipboard.
  • Options: Comments includes rung comments in the diff. Tag Values includes initial tag values.
  • Visual / Raw Text toggle switches all rung cards between diagram and text modes without rerunning the comparison.
Tip: The diff detects changes in rung content even when rung count is identical — catches edits that only changed rung text or instructions.

💡 Tips & Notes

  • An unsaved-change indicator (*) appears on the editor tab label whenever there are uncommitted edits.
  • Closing or refreshing the browser tab will prompt you before any unsaved changes are lost.
  • Comparison files (.cmpjson) can be re-opened with Open when a Compare tab is active.
  • Right-click any tag member row in the editor to quickly copy its name or current value to clipboard.
  • All processing is local — nothing is sent over the internet. The app runs entirely on your machine.
  • Use the organizer search box to filter tree nodes by name without fully expanding the tree.
